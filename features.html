Preprocessing¶

leabratf.features.preprocessing.remove_nan_inf(df, reindex=True)¶

Removes all rows that have NaN, inf or -inf as a value, and then optionally reindexes the dataframe.

Parameters:
  • df (pd.DataFrame) – Dataframe to remove NaNs and Infs from.
  • reindex (bool, optional) – Reindex the dataframe so that there are no missing indices.
Returns:

df – Dataframe with all the NaNs and Infs removed.
Return type:

pd.DataFrame
